# 🎵 Spotify Clone

A responsive Spotify-inspired web application built using **HTML5** and **CSS3**. This project recreates the look and feel of Spotify's web player, featuring a sidebar, navigation bar, music cards, and a music player interface. It was developed to strengthen frontend development skills and responsive web design.

## 🚀 Live Demo

🔗 https://music-stream-ui.vercel.app/

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Spotify-inspired user interface
- Sidebar navigation
- Sticky navigation bar
- Music cards with hover effects
- Music player interface
- Album information section
- Volume and playback controls
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Montserrat)
- Font Awesome

## 💻 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Nishthaparve/Spotify-Clone.git
```

### Navigate to the Project Directory

```bash
cd Spotify-Clone
```

### Run the Project

Open the `index.html` file in your browser or use the **Live Server** extension in Visual Studio Code.
